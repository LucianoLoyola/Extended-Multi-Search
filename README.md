# Extended Multi-Search Extension

This Chrome extension extends the native "Ctrl+F" functionality, allowing you to search for multiple terms simultaneously with different highlighting colors.

## Installation

1. Clone the repository
2. Open Google Chrome.
3. Navigate to `chrome://extensions/`.
4. Enable **Developer mode** (toggle in the top right corner).
5. Click **Load unpacked**.
6. Select this folder (`Extended-Multi-Search`).

## Usage

1. Open any web page (refresh existing tabs after installation).
2. Press `Ctrl+F` (or `Command+F` on macOS).
3. The custom search bar will appear in the top right.
4. Type your search term.
5. Click **+ Add Search Box** to add another term.
